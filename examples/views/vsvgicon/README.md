### 介绍

适用iconfont字体图标库

### 基础用法

```html
<v-svg-icon name="icon-copyright"/>
```

## API

### Base Props

| 参数   | 说明           | 类型      | 默认值 |
| ------ | -------------- | --------- | ------ |
| name | 字体name  | _string_    | _ |
| classPrefix | 前缀 | _string_  | _    |
| size | 图标大小 | _string \| number_  | _    |
